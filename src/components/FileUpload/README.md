# File Upload component

A reusable React component for upload multiple files with drag and drop feature.

# Usage:

```js
<FileUpload label = "Browse Files", max_files = 5, imagePreview = false>
```

# **_Basic Props_**

- label (string):- The text to display on the input field. Default:"Browse Files".
- max_files (number):- Limit to add file at a time, Default:5.
- imagePreview:- It is a flag to show imagw preview, Default:false.
